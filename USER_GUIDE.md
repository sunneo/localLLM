# USER_GUIDE

本文件說明如何使用 localLLM 系統。

## 1. 啟動主程式

在 `bin/` 目錄下執行主程式：

```bash
python3 bin/chatcall.py
```
或直接輸入需求：
```bash
python3 bin/chatcall.py "請幫我分析這段程式碼 ..."
```

## 2. 主要功能
- 支援多模型 LLM 本地推論（Qwen2.5 系列）。
- 可自動規劃任務並串接多工具。
- 支援 RAG（檢索增強生成）知識查詢與互動評分。
- 可讀取檔案、分析程式、生成程式碼。

## 3. 常用工具
- `text_reader`：讀取檔案內容。
- `code_analyzer`：分析程式邏輯。
- `write_code`：根據需求生成程式碼檔案。
- `chatter`：進行一般聊天。

## 4. 使用範例
- 讀取檔案：
  > 請讀取 /path/to/file.txt
- 分析程式：
  > 請分析這段 Python 程式碼 ...
- 生成程式：
  > 請幫我寫一個 bubble sort 的 C 程式

## 5. 對話與任務歷史
- 系統會自動保存最近 15 筆對話與任務。

## 6. 進階
- 可根據需求自動串接多工具，並進行多輪任務接力。
- 支援多使用者個人化資料。
